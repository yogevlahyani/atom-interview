# Atom interview
This repository contains a solution to an interview coding challenge focused on GPS trajectory data analysis. The tasks include data cleaning, trajectory plotting, extracting valuable time windows based on speed and acceleration, and a bonus task for pattern recognition.

## Install requirements
To install the required packages, run:
```bash
pip install -r requirements.txt
```

## Run the main analysis (Task 1 & 2)
To run the main analysis, execute the following command in your terminal:

```bash
python main.py
```

## Run the pattern recognition analysis (Task 3 - Bonus)
To perform pattern recognition on the trajectory data, run:
```bash
python pattern_recognition.py
```
